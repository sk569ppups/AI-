# AI Prompt Chart Beginner

GPTで新しいチャットを始める際に使う、初心者向けのプロンプト作成補助ツールです。  
やりたいこと・用途・出し方を選ぶことで、そのまま貼って使えるチャット用プロンプトを作成できます。

## 概要
このツールは、AIを使いたいが「最初に何をどう頼めばいいか分からない」という人向けに、GPTで使うチャット用プロンプトの土台を作るための入口ツールです。  
選択式で進めることで、用途に合ったチャットの方向性とプロンプトを整理できます。

## できること
- GPTで使うチャット用プロンプトの作成
- 用途に応じたチャットの方向性整理
- 初心者向けのAI活用入口支援
- プロンプト設計の基本構造の理解補助

## 想定している使い方
- 画像やPOPの方向性整理
- SNS用文章の作成
- 文章の校正や整え
- レポートの要点整理や読み解き
- 問題表現や内容確認
- 研修内容の振り返り

## 使い方
1. やりたいことを選ぶ  
2. 用途を選ぶ  
3. 出し方を選ぶ  
4. 表示されたプロンプトをコピーする  
5. GPTの新しいチャットに貼り付けて使う  

## 特徴
- GPTのみで完結するため導入しやすい
- 選択式なので初心者でも使いやすい
- チャットの役割やプロンプトの方向性を整理しやすい
- そのまま使える実務向けのたたき台を作りやすい
- プロンプト設計の入口ツール兼、学習用ツールとして使える

## 注意事項
- 本ツールは、GPTを使ったチャット開始を補助するためのものです
- 生成結果の最終確認・判断は、利用者ご自身で行ってください
- スマホ利用時は、GPTリンクがブラウザで開く場合があります

## 免責

OpenAIによる監修・保証・推奨を示すものではありません。  
生成結果の内容・正確性・適合性は利用環境や入力内容により変動するため、最終確認・判断は利用者自身で行ってください。
